# Antibody Hackathon Project

This repository contains the submissions for the Antibody Hackathon challenges.

## Structure

- **Challenge1/**: Contains metrics, sequences, and structures for Challenge 1.
- **Challenge2/**: Contains metrics, sequences, and structures for Challenge 2.
- **pitch/**: Contains the presentation materials.

## Contents

Each challenge directory contains:
- `metrics/`: Summary text files of metrics.
- `sequences/`: FASTA files for the designs.
- `structures/`: PDB files of the complexes and PAE JSON files.
